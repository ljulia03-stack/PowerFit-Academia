# PowerFit-Academia
A PowerFit Academia é um espaço moderno e completo dedicado à sua saúde, bem-estar e performance física. Aqui você encontra equipamentos de última geração, treinos personalizados e uma equipe de profissionais altamente qualificados prontos para te ajudar a alcançar seus objetivos.
# 🏋️‍♂️ PowerFit Academia - Website Oficial

Este é o repositório do site institucional da **PowerFit**, projetado com um design moderno, focado em conversão e totalmente responsivo (adaptado para celulares e computadores).

## 🎨 Identidade Visual
* **Predominante:** Preto e Cinza Espacial (`#0b0b0b`, `#161616`)
* **Destaque:** Laranja Neon/Fogo (`#ff4500`)

## 🛠️ Tecnologias
* HTML5 (Estrutura semântica)
* CSS3 (Estilização avançada e Flexbox)
* Font Awesome (Ícones integrados)

## 🚀 Como Visualizar
Basta clonar o repositório e abrir o arquivo `index.html` em qualquer navegador!
